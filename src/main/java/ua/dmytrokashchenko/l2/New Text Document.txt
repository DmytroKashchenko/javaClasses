Equals and hashCode  описать словестно контракт (выполнить письменно).

Однакові об'єкти мають однаковий хешкод, який є незмінним. Метод Equals для однакових об'єктів повертає значення true. Якщо хеш код об'єктів однаковий - то це не означає що об'єкти однакові(пояснюється це можливою наявністю колізій при генерації хеш коду).

Какие методы есть в классе Object и для чего они используются (выполнить письменно).

clone() - створення копії об'єкту на основі вже існуючого об'єкту - реалізовано на низькому рівні в jvm

equals(Object obj) - метод для порівняння об'єкту, рекомендується перевизначати його в своїх класах, по замовчуванню порівнює посилання на об'єкти

finalize() - метод, який може бути викликаний при видаленні об'єкту

getClass() - повертає клас об'єкта

hashCode() - повертає хеш код об'єкта

notify(), wait() - поновлення і очікування потока

toString() - повертає строкове представлення об'єкта. По замовчуванню виводиться ім'я классу + хеш код об'єкту
